# 🌦️ SkyCast: Real-Time Weather App
* STILL BUILDING

A sleek, responsive weather application that provides real-time meteorological data for any city in the world. Built with modern web technologies and powered by the **OpenWeatherMap API**.

---

## ✨ Features
* **Global Search:** Get current weather data for any city worldwide.
* **Dynamic Backgrounds:** UI colors and icons change based on weather conditions (e.g., sunny, rainy, snowy).
* **Detailed Metrics:** View temperature, humidity, wind speed, and "feels like" conditions.
* **Auto-Location:** Optional geolocation support to fetch weather for your current position.
* **Responsive Design:** Fully optimized for mobile, tablet, and desktop views.

---

## 🛠️ Tech Stack
* **Frontend:** React.js / Vue.js / Vanilla JS
* **Styling:** Tailwind CSS / CSS3
* **API:** [OpenWeatherMap API](https://openweathermap.org/api)
* **HTTP Client:** Axios / Fetch API

---

## 🚀 Getting Started

### 1. Prerequisites
You will need an API key from OpenWeatherMap.
1. Go to [OpenWeatherMap](https://openweathermap.org/) and create a free account.
2. Navigate to the **API Keys** tab in your dashboard.
3. Generate a new key and copy it.

### 2. Installation
Clone the repository and install the dependencies:
